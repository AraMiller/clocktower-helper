### 场景三：被毒的双子 (The Poisoned Twin) ❌

**设置**：
- Evil Twin 被 Poisoner 投毒（`isPoisoned = true`）
- 恶魔死亡

**预期行为**：
- 中毒的 Evil Twin 应该失去"阻止好人胜利"的能力
- 好人应该立即获胜

**代码分析**：
```typescript
// app/engine.ts 第 155-166 行
if (deadDemon && !aliveDemon) {
  if (evilTwinPair) {
    const evilTwin = seats.find(s => s.id === evilTwinPair.evilId);
    const goodTwin = seats.find(s => s.id === evilTwinPair.goodId);
    const bothAlive = evilTwin && !evilTwin.isDead && goodTwin && !goodTwin.isDead;
    
    if (bothAlive) {
      // 恶魔死亡但双子都存活，善良无法获胜，游戏继续（返回 null）
      return null;
    }
  }
  ...
}
```

**问题**：
- ❌ 代码只检查了 `bothAlive`（双子是否都存活）
- ❌ **没有检查 Evil Twin 是否中毒或醉酒**
- ❌ 根据游戏规则，中毒的角色会失去能力
- ❌ 如果 Evil Twin 中毒，即使两个双子都存活，也不应该阻止好人胜利

**修复方案**：
需要在检查 `bothAlive` 时，同时检查 Evil Twin 是否中毒或醉酒。如果 Evil Twin 中毒或醉酒，其"阻止好人胜利"的能力失效。

---

## 修复代码

已修改 `app/engine.ts` 的 `checkGameOver` 函数，在第 162-166 行添加了对 Evil Twin 中毒/醉酒状态的检查。

**修复后的代码**：
```typescript
// 检查 Evil Twin 是否中毒或醉酒（中毒/醉酒的角色会失去能力）

}
```

**修复验证**：
- ✅ 如果 Evil Twin 中毒，即使两个双子都存活，也不会阻止好人胜利
- ✅ 如果 Evil Twin 醉酒，同样不会阻止好人胜利
- ✅ 只有当 Evil Twin 健康且两个双子都存活时，才会阻止好人胜利
